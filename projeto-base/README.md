# Projeto Avaliativo - Desenvolvimento Web

## Tema: Portal Turístico Interativo – "Descubra o Brasil"

Este projeto foi desenvolvido para a disciplina de Desenvolvimento Web, utilizando HTML5 semântico e CSS externo.

### Estrutura do Projeto
- `index.html`: Página inicial
- `destinos.html`: Página de destinos turísticos
- `gastronomia.html`: Página de gastronomia brasileira
- `contato.html`: Página de contato
- `css/style.css`: Arquivo de estilos CSS
- `img/`: Pasta com imagens do projeto
- `README.md`: Este arquivo

### Tecnologias Utilizadas
- HTML5
- CSS3 (com Flexbox)

### Como Executar
Abra qualquer arquivo .html em um navegador web.
